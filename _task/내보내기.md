# Export 양식 개선 및 한글 인코딩 수정

## 목표

1. [source.xlsx](file:///c:/big20/live_meeting/source.xlsx) 템플릿 형식에 맞춰 내보내기 기능 개선
2. AI 회의록 요약을 활용하여 회의 목적, 주요 내용, 결론 및 향후 계획 자동 작성

## 현재 상황

### source.xlsx 템플릿 구조
```
회의명          | [값]
회의 유형       | [값]
회의일시        | [값]
참석자          | [값]
작성자          | [값]
회의 목적       | [값]
주요 내용       | [값]
결론 및 향후 계획 | [값]
```

### 데이터 소스
- **메타데이터**: Meeting 모델 (title, meeting_type, meeting_date, attendees, writer)
- **AI 요약**: Summary 모델 (content 필드에 AI 생성 요약 저장)
- **전사 내용**: Transcript 모델

## 구현 계획

### 1. Export API 수정 (export.py)

#### 변경 사항

1. **메타데이터 조회 추가**
   - Meeting 정보에서 메타데이터 가져오기
   - Summary 정보 조회 (AI 요약)

2. **AI 요약 파싱**
   - Summary.content를 파싱하여 회의 목적, 주요 내용, 결론 추출
   - 파싱 실패 시 기본값 또는 전체 요약 사용

3. **source.xlsx 템플릿 기반 Excel 생성**
   - openpyxl로 source.xlsx 로드
   - 각 필드에 데이터 채우기
   - 새 파일로 저장하여 반환

4. **한글 인코딩 수정**
   - CSV: `encoding='utf-8-sig'` 사용 (BOM 추가로 Excel에서 한글 정상 표시)
   - Excel: openpyxl은 기본적으로 UTF-8 지원

## 검증 계획

1. **한글 인코딩 테스트**
   - CSV 다운로드 후 Excel에서 열기
   - 한글이 정상적으로 표시되는지 확인

2. **템플릿 형식 테스트**
   - Excel 다운로드 후 source.xlsx와 동일한 형식인지 확인
   - 모든 필드가 올바르게 채워졌는지 확인

3. **AI 요약 파싱 테스트**
   - 다양한 AI 요약 형식에 대해 파싱이 정상 작동하는지 확인
   - 파싱 실패 시 fallback 동작 확인
