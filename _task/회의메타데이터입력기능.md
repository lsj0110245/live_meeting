## 사용자 플로우

### 실시간 녹음

1. 사용자가 "녹음 시작" 버튼 클릭
2. 메타데이터 입력 모달 표시 (회의일시는 현재 시간으로 자동 설정)
3. 모든 필드 입력 후 "확인" 클릭
4. 메타데이터가 WebSocket으로 서버에 전송되고 Meeting 생성
5. 실제 녹음 시작
6. 녹음 종료 시 "수정하시겠습니까?" 확인 모달 표시
7. 수정 선택 시 메타데이터 편집 모달 표시, 그대로 저장 선택 시 대시보드로 이동

### 파일 업로드

1. 사용자가 "파일 업로드" 버튼 클릭하여 파일 선택
2. 메타데이터 입력 모달 표시 (회의일시 직접 입력 가능)
3. 모든 필드 입력 후 "확인" 클릭
4. 메타데이터와 파일이 함께 서버로 전송
5. 서버에서 Meeting 생성 및 백그라운드 STT 처리 시작

## 기술적 세부사항

### WebSocket 메타데이터 전송 형식

```javascript
{
    type: 'metadata',
    data: {
        title: '회의명',
        meeting_type: '회의유형',
        meeting_date: '2026-02-02T16:00',  // ISO 8601 format
        attendees: '홍길동, 김철수',
        writer: '작성자명'
    }
}
```
