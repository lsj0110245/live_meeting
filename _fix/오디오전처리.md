# 오디오 전처리 고도화 계획 (Denoise & Normalize)

## Goal Description
작은 목소리의 인식률을 높이고자 단순히 볼륨을 키울 경우 배경 잡음(Noise)까지 같이 커져 인식률이 오히려 떨어지는 문제를 해결합니다. 
**"잡음 제거(Denoise) 후 증폭(Normalize)"** 이라는 2단계 프로세스를 도입하여 STT 엔진에 가장 깨끗하고 명확한 음성 데이터를 전달하는 것이 목표입니다.

## User Review Required
> [!IMPORTANT]
> **라이브러리 추가**: `noisereduce`, `scipy` 패키지 설치가 필요합니다.
> **레이턴시**: 실시간 처리 시 잡음 제거 연산으로 인해 약 **0.1~0.3초의 지연**이 발생할 수 있습니다. 이는 "정확도"를 위한 투자입니다.
